# owl2jkgen translator script

The scripts in this directory use the PheKnowLator PyPi package (pkt-kg) 
to translate OWL files into **JKG Edge/Node** (JKGEN) format.

**owl2jkgen.py** is the latest version.

# Script function
The **owl2jkgen** application:
1. downloads an OWL file from a specified URL using _wget_
2. uses _rdflib_ to convert the OWL data into a rdflib graph
3. uses the PheKnowLator package (_pkt-kg_) to convert the rdflib graph into a OWL-NETS graph
4. writes data from the OWL-NETS graph to files in JKGEN format

## OWL formats
The **owl2jkgen** application translates OWL files in the following serializations:
1. RDF/XML
2. Turtle

OWL files in other serializations (e.g., OWL/RDF, NTriples) will require conversion to RDF/XML.

## Script dependencies
1. **owl2jkgen** downloads **owltools**, a JAR file in the _pkt_kg_ directory.
2. The script uses classes and functions from the _/utilities_ path.
