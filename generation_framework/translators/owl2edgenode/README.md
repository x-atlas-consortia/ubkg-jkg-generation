# owl2edgenode translator script

The scripts in this directory use the PheKnowLator PyPi package (pkt-kg) 
to translate OWL files into OWLNETS format.

**owl2edgenode.py** is the latest version.

This script explicitly downloads the JAR file in the _pkt_kg_ directory.

# Archive
The _archive_ directory contains earlier versions of scripts.

## fix_tsv_file.py
This script was once used to post-process the node file generated by PheKnowlator. 
It appears to no longer be necessary.
